# moonPlayer 开发任务

## 项目结构

```
moonplayer/
├── server/                    # 后端 (Fastify + SQLite)
│   ├── src/
│   │   ├── routes/           # API 路由
│   │   ├── db/               # 数据库
│   │   ├── services/         # 业务逻辑
│   │   └── index.ts
│   └── package.json
├── web/                       # 前端 (React + Vite)
│   ├── src/
│   │   ├── components/       # UI 组件
│   │   ├── stores/           # Zustand 状态
│   │   └── App.tsx
│   └── package.json
└── shared/                    # 共享类型
    └── types.ts
```
